## 📋 Quick Reference

### Start Here
```
1. Open: QUICKSTART.md
2. Run: setup.ps1 or setup.bat
3. Create Supabase account
4. Configure: server/.env
5. Run: npm run dev (both folders)
6. Open: http://localhost:5173
7. Enjoy your portfolio! 🎉
```

### Edit Content
```
1. Go to Supabase Dashboard
2. Editor → sections table
3. Update title, content, image_url
4. Refresh portfolio (F5)
5. Done! Changes appear instantly
```

### Deploy
```
1. Run: npm run build (in client)
2. Deploy dist/ to Vercel/Netlify
3. Deploy server/ to Railway/Render
4. Update API URL in frontend
5. Your portfolio is now live!
```
